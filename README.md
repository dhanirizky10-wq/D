# Marketplace Makanan Lokal

Web marketplace sederhana untuk makanan lokal Indonesia menggunakan Flask.

## Fitur
- Daftar produk makanan lokal
- Detail produk
- Keranjang belanja
- Checkout sederhana

## Instalasi
1. Install dependencies: `pip install -r requirements.txt`
2. Jalankan app: `python app.py`
3. Buka browser ke http://127.0.0.1:5000/

## Struktur
- `app.py`: Aplikasi utama Flask
- `templates/`: Template HTML
- `static/`: CSS dan gambar
